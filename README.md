# Projeto Next.js - 50 Exercícios

Esse é um projeto feito com Next.js contendo 50 exercícios. Fazer exercício é bom para aumentar o músculo, mas será que meu cérebro fica maior?

## Descrição

Este repositório contém uma série de exercícios desenvolvidos usando o framework Next.js. Cada exercício aborda diferentes aspectos e funcionalidades do Next.js, proporcionando uma maneira prática de aprender e dominar a ferramenta.

## Exercícios

1. Crie uma Página Simples
   - Crie uma nova página `about.js` no diretório `pages` e exiba um texto simples nela.<br><br>

2. Link Entre Páginas
   - Crie uma página `contact.js` e adicione um link na página principal para redirecionar para `contact`.<br><br>

3. Estilização com CSS Modules
   - Adicione estilização utilizando CSS Modules em uma página existente.<br><br>

4. Utilize Tailwind CSS
   - Configure e aplique Tailwind CSS no seu projeto Next.js.<br><br>

5. Componentes Personalizados
   - Crie um componente `Header` e utilize-o em várias páginas do seu projeto.<br><br>

6. Configurar Rotas Dinâmicas
   - Crie páginas dinâmicas utilizando rotas dinâmicas como `[id].js` no Next.js.<br><br>

7. Consumir uma API REST
   - Faça uma requisição a uma API REST e exiba os dados em uma página.<br><br>

8. Utilize `getStaticProps`
   - Use `getStaticProps` para fazer pré-renderização com dados estáticos.<br><br>

9. Utilize `getServerSideProps`
   - Use `getServerSideProps` para fazer pré-renderização com dados dinâmicos.<br><br>

10. Incremental Static Regeneration (ISR)
    - Configure ISR para páginas que precisam ser atualizadas após o build inicial.<br><br>

11. Usar `useEffect` e `useState`
    - Crie uma página que utilize hooks `useEffect` e `useState` para gerenciar estado e efeitos colaterais.<br><br>

12. Autenticação com NextAuth.js
    - Implemente autenticação utilizando NextAuth.js em um projeto Next.js.<br><br>

13. Deploy com Vercel
    - Realize o deploy do seu projeto Next.js na plataforma Vercel.<br><br>

14. Middleware de Autenticação
    - Crie um middleware para proteger rotas e garantir que o usuário esteja autenticado.<br><br>

15. Internationalization (i18n)
    - Configure a internacionalização (i18n) no seu projeto Next.js para suportar múltiplos idiomas.<br><br>

16. Redirecionamentos e Rewrites
    - Configure redirecionamentos e rewrites no arquivo `next.config.js`.<br><br>

17. Imagem Otimizada com `next/image`
    - Utilize o componente `next/image` para otimização de imagens.<br><br>

18. GraphQL com Apollo Client
    - Implemente GraphQL no seu projeto Next.js utilizando Apollo Client.<br><br>

19. Criar API Routes
    - Crie rotas de API dentro do diretório `pages/api` e faça uma requisição a partir do cliente.<br><br>

20. SSR com Apollo Server
    - Configure o servidor Apollo para fazer SSR (Server-Side Rendering) de consultas GraphQL.<br><br>

21. Configurando Paths Absolutos
    - Configure paths absolutos para importações mais limpas e organizadas.<br><br>

22. Dark Mode com Tailwind CSS
    - Implemente um switch de Dark Mode utilizando Tailwind CSS.<br><br>

23. Usar Context API
    - Use Context API para gerenciar estado global em seu projeto Next.js.<br><br>

24. Configurar PWA
    - Configure seu projeto Next.js como Progressive Web App (PWA).<br><br>

25. Adicionar Typescript
    - Adicione suporte a TypeScript ao seu projeto Next.js.<br><br>

26. Formulários com Formik e Yup
    - Implemente formulários utilizando Formik para gerenciamento de estado e Yup para validação.<br><br>

27. Publicando um Blog
    - Crie um simples sistema de blog onde você pode adicionar, editar e excluir posts.<br><br>

28. Componentização de Layout
    - Crie componentes de layout reutilizáveis para cabeçalho, rodapé e sidebar.<br><br>

29. Carregamento Lazy com React.lazy
    - Implemente carregamento lazy de componentes usando `React.lazy`.<br><br>

30. Sistema de Comentários
    - Adicione um sistema de comentários a um post de blog.<br><br>

31. Data Fetching com SWR
    - Utilize o SWR para fazer data fetching eficiente em seu projeto Next.js.<br><br>

32. Acessibilidade (A11y)
    - Melhore a acessibilidade do seu site aplicando práticas recomendadas.<br><br>

33. Usando Storybook
    - Configure e use Storybook para desenvolver e testar componentes de forma isolada.<br><br>

34. Deploy com Docker
    - Crie uma imagem Docker para seu projeto Next.js e faça o deploy utilizando Docker.<br><br>

35. Hot Module Replacement (HMR)
    - Configure HMR para desenvolvimento mais eficiente.<br><br>

36. Teste de Unidade com Jest
    - Implemente testes de unidade usando Jest.<br><br>

37. Teste de Integração com Cypress
    - Implemente testes de integração utilizando Cypress.<br><br>

38. Usando Redux
    - Adicione Redux para gerenciar o estado global do seu projeto.<br><br>

39. Performance Optimization
    - Otimize a performance do seu site utilizando Lighthouse e outras ferramentas.<br><br>

40. Visualização de Dados com D3.js
    - Implemente visualizações de dados utilizando D3.js.<br><br>

41. Usar Framer Motion
    - Adicione animações interativas ao seu projeto com Framer Motion.<br><br>

42. Multi-Step Forms
    - Crie formulários de múltiplos passos com validação em cada etapa.<br><br>

43. Paginação de Dados
    - Implemente paginação para grandes conjuntos de dados.<br><br>

44. Filtro e Busca
    - Adicione funcionalidades de filtro e busca para listar itens dinamicamente.<br><br>

45. Upload de Arquivos
    - Implemente upload de arquivos e armazenamento utilizando um serviço como Firebase Storage.<br><br>

46. Gestão de Dependências
    - Utilize Yarn Workspaces ou Lerna para gerenciar dependências em monorepo.<br><br>

47. Responsive Design
    - Crie um layout responsivo que funcione bem em todos os dispositivos.<br><br>

48. Custom Document
    - Personalize o componente `Document` para adicionar metadados, fontes ou outras configurações globais.<br><br>

49. Prevenção de Duplicação de Conteúdo
    - Configure cabeçalhos HTTP e meta tags para evitar duplicação de conteúdo e melhorar SEO.<br><br>

50. Usando `useReducer`
    - Utilize o hook `useReducer` para gerenciar estado complexo em seu projeto.<br><br>
